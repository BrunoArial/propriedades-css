PROPRIEDADES DO CSS

Exemplos de propriedades CSS

• Propriedades de Texto e Fonte

• Color

A propriedade 'color' define a cor do texto de um elemento HTML, pode ser definido por valor hexadecimal (HEX), RGB, nome da cor e todos outros sistemas de cores existentes

Este é um exemplo de texto definido como 'blue'

[index.html]


<p class="azul">Este é um exemplo de texto definido como 'blue'</p>

[style.css]


.azul {
    color: blue;
}

• Font-family (Família da fonte)

A propriedade 'Font-family' especifica a fonte a ser usada para o texto

Este é um exemplo de texto definido com a fonte 'Times new roman'

[index.html]


<p class="fonte">Este é um exemplo de texto definido com a fonte 'Times new roman'</p>

[style.css]


.fonte {
    font-family: "Times New Roman", Times, serif;
}

• Font-size

A propriedade 'Font-size' controla o tamanho do texto, pode ser definido em pixels, ems, rems, porcentagem, etc.

Este é um exemplo de texto definido com o tamanho de 10 pixels

Este é um exemplo de texto definido com o tamanho de 30 pixels

[index.html]


<p class="fontsize">Este é um exemplo de texto definido com o tamanho de 10 pixels</p>
<p class="fontsize2">Este é um exemplo de texto definido com o tamanho de 30 pixels</p>

[style.css]


.fontsize {
    font-size: 10px;
}
.fontsize2 {
    font-size: 30px;
}

• Text-align (Alinhamento de Texto)

A propriedade 'text-align' controla o alinhamento horizontal do texto dentro de seu elemento container. Os valores mais comuns são left (esquerda), center (centro), right (direita) e justify (justificado).

Este texto está centralizado com text-align: center;

[index.html]


<p class="exemplo-center">Este texto está centralizado com 'text-align: center;'</p>

[style.css]


.exemplo-center {
    text-align: center;
}

• Propriedades de Fundo (Background)

• Background-color (Cor de Fundo)

A propriedade 'background-color' define a cor de fundo de um elemento. Pode ser definida por valor hexadecimal (HEX), RGB, nome da cor, entre outros.

Este texto está com fundo amarelo usando background-color: yellow;

[index.html]

<p class="exemplo-bg">Este texto está com fundo amarelo usando 'background-color: yellow;'</p>

[style.css]


.exemplo-bg {
    background-color: yellow;
}

• Propriedades de Caixa e Posicionamento

• Margin (Margem Externa)

A propriedade 'margin' define o espaço externo ao redor de um elemento.

Este parágrafo possui uma margem de 40px.

[index.html]

<p class="exemplo-margin">Este parágrafo possui uma margem de 40px.</p>

[style.css]

.exemplo-margin {
    margin: 40px;
}

• Padding (Preenchimento Interno)

A propriedade 'padding' define o espaço interno entre o conteúdo e a borda de um elemento.

Este parágrafo possui um padding de 30px.

[index.html]

<p class="exemplo-padding">Este parágrafo possui um padding de 30px.</p>

[style.css]

.exemplo-padding {
    padding: 30px;
    background: #e0e0e0;
}

• Border (Borda)

A propriedade 'border' define a borda ao redor de um elemento.

Este parágrafo possui uma borda de 2px sólida azul.

[index.html]

<p class="exemplo-border">Este parágrafo possui uma borda de 2px sólida azul.</p>

[style.css]

.exemplo-border {
    border: 2px solid blue;
    padding: 10px;
}

• Width e Height (Largura e Altura)

As propriedades 'width' e 'height' definem a largura e altura de um elemento.

Este bloco tem 200px de largura e 60px de altura.

[index.html]

<div class="exemplo-wh">Este bloco tem 200px de largura e 60px de altura.</div>

[style.css]

.exemplo-wh {
    width: 200px;
    height: 60px;
    background: #b3d9ff;
    text-align: center;
    line-height: 60px;
}

• Display (Exibição)

A propriedade 'display' controla como um elemento é exibido na página. Os valores mais comuns são block, inline e inline-block.

Span 1 Span 2

[index.html]

<span class="exemplo-inline">Span 1</span>
<span class="exemplo-inline">Span 2</span>

[style.css]

.exemplo-inline {
    display: inline-block;
    background: #ffd966;
    padding: 5px 15px;
    margin: 5px;
    border-radius: 5px;
}

• Position (Posicionamento) e Top, Right, Bottom, Left

A propriedade 'position' define o método de posicionamento de um elemento. Os valores mais comuns são static, relative, absolute e fixed. As propriedades top, right, bottom e left ajustam a posição do elemento quando position é diferente de static.

Este bloco está posicionado 20px da esquerda e 10px do topo.

[index.html]

<div class="exemplo-pos-container">
    <div class="exemplo-pos">Este bloco está posicionado 20px da esquerda e 10px do topo.</div>
</div>

[style.css]

.exemplo-pos-container {
    position: relative;
    height: 80px;
    background: #f0f0f0;
}
.exemplo-pos {
    position: absolute;
    top: 10px;
    left: 20px;
    background: #ffb3b3;
    padding: 10px 20px;
    border-radius: 5px;
}
